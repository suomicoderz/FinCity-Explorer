# FinCity-Explorer
FinCity Explorer is a Python command-line tool that transforms raw Finnish city data into actionable insights. Whether you're planning a Nordic road trip or studying regional demographics, this tool lets you locate the nearest city to any coordinates, compute distances between urban centers, filter cities by population, and highlight capitals. It harnesses the haversine formula for accurate geodesic calculations and presents clean output. With simple commands, explore Finland’s five major cities—Helsinki, Tampere, Espoo, Turku, and Vantaa—by region, size, or proximity. Ideal for travellers and geographers, FinCity Explorer turns static tables into a dynamic urban atlas.
